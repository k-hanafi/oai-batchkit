# batchkit-frontend

Vite + React + TypeScript + Tailwind. The renderer process for the Electron shell, and the source of all UI.

## Local setup

```bash
cd frontend
npm install
npm run dev
```

## Common commands

```bash
npm run dev          # vite dev server with HMR
npm run build        # type-check + production build to dist/
npm run lint         # eslint
npm run typecheck    # tsc -b --noEmit
npm run test         # vitest run
npm run format       # prettier --write
```
